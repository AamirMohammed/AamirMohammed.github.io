VRMania

A platform hub for the arcade system to handle both internally made and externally licensed games . Created for the HTC Vive

Responsibilities Include

  • Development of a complete software from its planning to release phase
  • Architecturing the platform using different design patterns
  • Project Management

Implementing core systems such as-

  • Games Launcher for launching the different games from inside the hub
  • Writing logic to communicate between the platform and different games for modifying settings and fetching score.
  • Writing a payment system to handle credits inserted by the customer
  • Writing a ticket dispensing system (redemption) to dispense tickets
  • Creating Installers for different games
  • Implementing in game UI and related systems
  • Completing additional features and tasks
